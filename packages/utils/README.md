# @aiql-org/utils

Shared file I/O helpers and format utilities for the AIQL SDK.

## Features

- `readAiqlFile(path)` — Read AIQL source files
- `writeOutput(path, content)` — Write transpiled output
- `discoverAiqlFiles(dir)` — Recursively find `.aiql` files
- `getTargetExtension(target)` — Get file extension for target
- `SUPPORTED_TARGETS` — List of all supported transpilation targets
