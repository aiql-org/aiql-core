# @aiql-org/security

Quantum-proof cryptography and trust infrastructure for AIQL.

## Features

- **Dilithium** — Post-quantum digital signatures
- **Kyber** — Post-quantum key encapsulation
- **Trust Registry** — Agent authentication and trust management

## Usage

```typescript
import { DilithiumSecurity, TrustRegistry } from '@aiql-org/security';
```
